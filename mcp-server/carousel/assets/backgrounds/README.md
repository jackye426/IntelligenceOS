# Optional per-template background images

Place JPEG/PNG files here named after template IDs:

- `classic_blue.jpg` (optional — uses solid color if missing)
- `photo_center_hook.jpg`
- `photo_body_left.jpg`
- `photo_body_right.jpg`
- `minimal_white.jpg`

Recommended size: 1080×1350 (4:5). The builder full-bleeds the image on hook/body/cta slides.
